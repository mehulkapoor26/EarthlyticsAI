# EarthlyticsAI 🌍

AI-powered carbon footprint tracking platform built during Hackowasp Hackathon at Thapar Institute under SDG-13: Climate Action.

## Features
- Electricity usage tracking
- Plastic consumption analysis
- Travel emission calculator
- Carbon intelligence dashboard
- AI sustainability insights
- Achievement badge system
- Interactive data visualizations

## Tech Stack
- HTML
- CSS
- JavaScript

## Theme
UN Sustainable Development Goal 13 — Climate Action

## Hackathon
Hackowasp Hackathon @ Thapar Institute
18–19 April

## Vision
Helping people understand and reduce their environmental impact through interactive technology and AI-powered insights.
